# ⚔ Dark Souls CRUD Web Service

Welcome to the land of Lordran — where death is a feature, not a bug, and every database record is one bad decision away from being soft-locked forever.

This web application is a full-stack CRUD interface themed around the brutally unforgiving world of **Dark Souls**. Built by **Ben & Jared** (Team 200), it lets you conjure, inspect, mutate, and banish the very entities that make Lordran tick: the haggard **Characters** who refuse to stay hollowed, the shadowy **Factions** vying for the age of fire, and the devastating **Weapons** used to settle every grudge in the most punishing ways imaginable.

On the front end, a dark, particle-drenched HTML/CSS/JavaScript client serves as your bonfire — a glowing hub where you fill out forms and fire off fetch requests like throwing embers into the void. On the back end, a PHP REST API stands guard like a fog-gate, routing your Create, Read, Update, and Delete requests to a MySQL database that holds all the lore.

Whether you want to register a new Undead warrior with a cursed greatsword and sworn allegiance to a sun-worshipping covenant, or wipe a treacherous faction clean from the records like the Painted World never existed — this app has you covered.
